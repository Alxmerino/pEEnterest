# pEEnterest

pEEnterest is a plugin that will generate a list of pins from the given board id.

## Usage
````
{exp:peenterest token="ACCESS_TOKEN" board_id="BOARD_ID" images="yes|no" creator="yes|no" counts="yes|no" media="yes|no"}
	{id}
	{link}
	{creator}
	{board}
	{created_at}
	{note}
	{color}
	{counts}
	{media}
	{attribution}
	{image}
	{metadata}
{/exp:peenterest}
````
